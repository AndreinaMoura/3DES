->LINGUAGENS DE PROGRAMAÇÃO: 
-->banco de dados: mysql, 
-->back-end: java, 
-->front-end: html, css, javascript, 
-->mobile: javascript, node js.

->tela de login e cadastro 
fluídas e de fácil entendimento nas cores preto, vermelho e amarelo.

tela de login:
logar com o email e senha,
perguntar se já possui cadastro e colocar um link para a tela de cadastro
um botão de login que leva para a tela principal.

tela de cadastro:
pedir o nome, endereço, CPF, data de nascimento, genêro, telefone, email, senha(um digite novamente)
botão enviar cadastro
após o cadastro bem sucedido redirecionar para a tela de login

tela principal:
mostrar a ficha do cliente(grupos musculares e pesos)
botao detalhar para mostrar status e frequência
mostar o menu lateral com a ficha do cliente, tela de pagamentos, atualizar dados

tela de pagamentos:
listar pagamentos feitos 
mostrar pagamentos próximos
mostrar pagamentos atrasados

tela de atualizar dados:
mostrar todos os dados e a opção de alterar individualmente
botao de atualizar
botão de deletar a conta/cadastro
